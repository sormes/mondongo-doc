Para ello tienes que mapearlos como archivos.

                'name' => 'string',

Al hacerlo se a침ade autom치ticamente el campo **file**, que es en el que se

    // como bytes

>[CAUTION]
>El archivo **no puede actualizarse**. El resto de campos mapeados si.
